Hi this is a short tutorial on how to use this Client,

Open Gui: RSHIFT
Set Bind: In the Gui or ".bind <module> <key>/NONE"
AccountManager: MultiPlayer -> Accounts -> Add online -> Paste -> Select Account
Bungeehack: MultiPlayer -> set Checkbox Bungeehack, to change fakeip: edit text under Checkbox
Disable SkinOverlay: RSHIFT -> HUD -> Skin
