Short tutorial on how to use SkillClient:

Open Gui: RSHIFT
Set Keybind: In the Gui or ".bind <module> <key>/NONE"
AccountManager: MultiPlayer -> Accounts -> Add online -> Paste -> Select Account
Bungeehack: MultiPlayer -> set Checkbox Bungeehack; to change IP: edit text under Checkbox
Disable SkinOverlay: RSHIFT -> HUD -> Skin

Please only download Skillclient from skillclient.com
SkillClient is free and it will stay like this forever.
